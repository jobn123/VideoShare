
# share

> video share page

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

# Params

*  id: 剧集id   index: 第几集
*  /?id=927&index=5  电视剧
*  /?id=927          电影
*  /variety?id=8&index=5  综艺
